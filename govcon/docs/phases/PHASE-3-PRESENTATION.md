# Phase 3: Client Proposal Presentation

## Objective
Generate a professional PowerPoint proposal deck that Still Mind Creative presents to Newport Wholesalers leadership. The deck must make the business case for entering government contracting, using real data from the FPDS/USASpending analysis already performed.

## Output
- File: `deliverables/presentation/newport-govcon-proposal.pptx`
- Format: 16:9 widescreen
- Target: 12-15 slides
- Audience: Newport Wholesalers ownership/leadership (non-technical business decision-makers)

## Design Direction
- Color palette: Ocean Gradient — Primary `065A82` (deep blue), Secondary `1C7293` (teal), Accent `21295C` (midnight), Text on dark: `FFFFFF`, Text on light: `1E293B`
- Font: Georgia (headers) / Calibri (body)
- Dark background for title + section dividers + closing. Light/white background for content slides.
- Visual motif: Bold stat callouts (large numbers) paired with clean supporting text
- NO generic clipart. Use shapes, icons, charts, and data visualizations.
- Every slide must have at least one visual element (chart, shape, icon, stat callout, or table)

### Slide 5: The Process — How Government Contracting Works
- **Background**: White
- **Headline**: "From Registration to Revenue: The Path Forward"
- **Layout**: Horizontal process flow (numbered steps with arrows)

  2. **Find** — Monitor contract opportunities daily via data feeds
  3. **Score** — Evaluate each opportunity with bid/no-bid framework
  4. **Bid** — Write proposal: capability statement, technical response, pricing
  5. **Win** — Deliver, invoice, build past performance for next contract
  6. **Scale** — Win rate improves with every contract. Past performance compounds.
- **Bottom text**: "The first 3-5 wins are the hardest. After that, the flywheel turns."
